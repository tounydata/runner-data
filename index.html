Conditions d'utilisation
```
1. Objet

Runner OS est une application personnelle d'analyse de performances en course à pied et trail. Elle est fournie à titre gratuit, sans garantie de disponibilité continue.

2. Données personnelles

Runner OS collecte et traite les données suivantes :

  • Données de compte : email (via Supabase Auth)
  • Données physiologiques : FC max, poids, taille, seuil lactique (saisies volontairement)
  • Données d'activité : activités Strava via l'API officielle Strava
  • Données de course : GPX, temps, stratégies (stockées en Supabase)

Ces données sont stockées dans Supabase (infrastructure européenne). Chaque utilisateur n'a accès qu'à ses propres données.

3. Données Strava

L'accès aux données Strava est effectué via l'API officielle Strava conformément aux conditions d'utilisation de l'API Strava. Les tokens d'accès sont stockés localement. Aucune donnée Strava n'est partagée avec des tiers.

4. Analyse IA

Les analyses de sortie utilisent l'API Groq (modèle llama). Les données d'activité sont transmises à Groq pour générer l'analyse. Aucune donnée d'identification personnelle (nom, email) n'est transmise.

5. Suppression des données

Tu peux supprimer ton compte et toutes tes données à tout moment en contactant l'administrateur de l'application. La déconnexion de Strava révoque l'accès immédiatement.

6. Responsabilité

Les conseils de stratégie, nutrition et FC fournis par Runner OS sont indicatifs. Ils ne remplacent pas l'avis d'un médecin ou d'un entraîneur professionnel. L'utilisateur reste seul responsable de ses choix d'entraînement.
